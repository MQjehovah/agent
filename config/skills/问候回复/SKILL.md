# 问候回复

## 概述

本技能用于处理用户的初始问候语（如“你好”、“嗨”等），以友好、热情的方式回应，并简要介绍自身身份或能力，从而建立良好的首次交互体验。

## 使用场景

- 用户发送类似“你好”、“嗨”、“hello”等问候语句时。
- 对话开始时，用户尚未提出具体问题或指令。
- 需要确立AI助手的身份并提供初步引导。

## 工作流

1. **接收用户输入**：获取 `{user_input}` 文本。
2. **识别问候意图**：检查 `{user_input}` 是否包含常见问候词（你好、嗨、hello、hi、早上好等）。若匹配，则继续；否则提示“本技能仅处理问候”。
3. **生成友好回应**：根据问候类型和语气，构造包含感谢、问候和简单自我介绍的回复。例如：“你好！很高兴见到你，我是你的AI助手，可以帮你解答问题、提供信息或完成任务。”
4. **输出结果**：返回完整的回应文本。

## 输出格式

直接输出一段自然语言文本，无需额外标记或结构化。

**示例输出**：
```
你好！很高兴见到你，我是你的智能助手，有什么我可以帮你的吗？
```
