# 安全审查清单

## 硬编码密钥
- [ ] 无 API Key、密码、Token 硬编码在源码中
- [ ] 敏感信息已移至环境变量或密钥管理服务
- [ ] `.env` 文件已加入 `.gitignore`

## SQL 注入
- [ ] 所有数据库查询使用参数化查询（Prepared Statements）
- [ ] 无字符串拼接构建 SQL
- [ ] ORM 使用时避免原生 SQL 中的拼接

## XSS 防护
- [ ] 用户输入在输出到页面时做了编码
- [ ] React 中避免使用 `dangerouslySetInnerHTML`
- [ ] 富文本内容使用安全的 HTML 清洗库

## CSRF 防护
- [ ] 状态变更请求使用了 CSRF Token
- [ ] Cookie 设置了 `SameSite` 属性
- [ ] 敏感操作要求二次确认

## 认证与授权
- [ ] 所有受限接口需要有效的认证凭证
- [ ] 权限检查在服务端执行（不仅在前端）
- [ ] 敏感接口遵循最小权限原则
- [ ] 会话超时和刷新机制正确

## 依赖漏洞
- [ ] 运行了 `npm audit` / `pip audit` / 等效工具
- [ ] 无高危或严重漏洞未处理
- [ ] 不使用的依赖包已移除

## 文件路径遍历
- [ ] 用户提供的文件名已做路径规范化
- [ ] 限制文件操作到白名单目录
- [ ] 禁止使用用户输入直接构造文件路径

## 命令注入
- [ ] 避免使用 `os.system`、`exec`、`eval` 等
- [ ] 使用子进程库时传参为数组而非字符串
- [ ] 用户输入不直接拼接为 shell 命令
