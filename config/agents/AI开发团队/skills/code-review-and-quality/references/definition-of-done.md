# 完成定义检查清单

## 测试
- [ ] 所有已有测试通过（绿线）
- [ ] 新增行为有对应的单元测试/集成测试
- [ ] Bug 修复包含回归测试，防止再次出现
- [ ] 测试覆盖边缘情况和异常路径

## 代码审查
- [ ] 代码已由至少一位同事审查通过
- [ ] 无严重或阻塞性问题未解决
- [ ] 所有审查意见均已回复或处理

## 文档
- [ ] API 文档已更新（接口签名、参数、返回值）
- [ ] 变更日志（CHANGELOG）已更新条目
- [ ] 如有必要，更新了 README 或内部 Wiki

## 安全
- [ ] 代码中无硬编码密钥、令牌、密码
- [ ] 用户输入已做校验和清洗
- [ ] 依赖项已扫描无已知漏洞
- [ ] 敏感操作有权限检查

## 性能
- [ ] 无 N+1 查询问题
- [ ] 列表接口已实现分页
- [ ] 未引入明显性能退化

## Git
- [ ] 提交是原子的（一个提交一个逻辑变更）
- [ ] 提交信息清晰描述变更原因
- [ ] 无调试代码、注释掉的代码、无关文件
