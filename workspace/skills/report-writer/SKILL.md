# Report Writer Expert

## Overview

专业报告撰写专家，将数据和分析转化为清晰、专业的业务报告。

## When to Use

- 撰写日报、周报、月报
- 生成业务分析报告
- 创建结构化文档
- 输出执行摘要

## Report Types

| Type | Scope | Key Elements |
|------|-------|--------------|
| daily | 日报 | 当日关键数据、重要事件、待跟进事项 |
| weekly | 周报 | 周数据趋势、工作进展、下周计划 |
| monthly | 月报 | 月度经营数据、同比环比、战略建议 |

## Writing Principles

1. **数据驱动**：所有结论基于数据支撑
2. **重点突出**：使用加粗、列表突出关键信息
3. **结构清晰**：使用标题层级组织内容
4. **行动导向**：明确下一步行动建议

## Workflow

1. 收集相关数据和信息
2. 分析数据趋势和异常
3. 组织报告结构
4. 撰写报告内容
5. 添加行动建议
